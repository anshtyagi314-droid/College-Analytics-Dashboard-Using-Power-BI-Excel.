Aspiring data analyst skilled in Excel, Power BI, SQL, and Python. Passionate about data visualization and building interactive dashboards for real-world insights. Experienced in education analytics projects like student performance and placement trends. Continuously learning, I aim to grow in data-driven roles and deliver, insight-based solutions.
verview

This project is a comprehensive College Analytics Dashboard built using Microsoft Excel and Power BI. It provides insights into student performance, faculty analytics, and placement trends through interactive and visually appealing dashboards.

🚀 Features
Admin Dashboard: Overview of students, faculty, departments, and fee statistics
Faculty Analytics: Workload, salary insights, qualifications, and experience analysis
Student Analytics: GPA, attendance, enrollment trends, and departmental distribution
Placement Analytics: Salary packages, placement trends, and company-wise hiring insights
Interactive Filters: Dynamic filtering by department, year, gender, and more
🛠️ Tools & Technologies
Microsoft Excel
Power BI
Data Visualization Techniques

Track student performance and attendance trends
Analyze faculty workload and salary distribution
Monitor placement success and salary packages
Identify department-wise performance patterns
